## 📋 What is svg2stencil?

svg2stencil helps you turn the paste layer of a printed circuit board (PCB) design into a 3D printable stencil. You start with a .svg file that shows where solder paste should go. Then, svg2stencil converts this into a custom .stl file ready for 3D printing. This lets you create accurate stencils for applying solder paste to PCBs in your own workshop or lab.

### Step 3: Open Your PCB Paste Layer File  

1. Launch svg2stencil by clicking its icon.  
2. Click **File** > **Open**, then select the `.svg` file of your PCB paste layer. This file should come from your PCB design software where you exported the paste layer.  
3. The app will load your design and prepare it for conversion.

---

### Step 4: Create Your 3D Stencil File

1. After your paste layer loads, you can adjust basic settings such as stencil thickness or print scale.  
2. Click the button labeled **Generate STL** or similar.  
3. Save the new `.stl` file to your desired folder. This file contains the 3D model ready to print.

---

### Step 5: Print Your Stencil  

1. Open the `.stl` file in your normal 3D printing software.  
2. Use your slicing program to prepare the file for printing.  
3. Print the stencil using your 3D printer. Use common materials like PLA or ABS plastic.  
4. After printing, clean the stencil if needed. It is now ready to use for solder paste application.

## 🔧 Troubleshooting and Tips  

- If svg2stencil does not open, check that your Windows version is supported and try running the app as Administrator.  
- If opening your SVG file fails, verify that it contains only the paste layer and is generated from your PCB's export settings.  
- For 3D print failures, confirm the STL file loads correctly in your slicing software and check printer setup.  
- Restart svg2stencil if it gets stuck or crashes mid-conversion.  
- Use PLA filament for easy printing and good stencil detail.  
- Avoid very thin stencil thicknesses below 0.5 mm to prevent print damage.
